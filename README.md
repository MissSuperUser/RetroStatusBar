Modded version of Super Status Bar, with a retro twist.

# How to use
Disable every system icon besides airplane mode, battery, network and wifi (keep battery percentage text if you really need it, even if it wasnt in AOSP Kitkat)

Next, Change the items order by making it so that the clock is next to the system icons

Then, Change system icons color to BFBFBF to make the clock and notifications colored

I also recommend enabling ticker text with custom colors for the good old notification ticker! 
# How to build
Use Apktool to build it, then sign and align the APK build. dunno how else to explain
